Books

Up 
get /v1/books
mengambil semua koleksi buku (collectionBooks)
Koleksi buku yang diambil dapat berasal dari pihak ketiga, atau berasal dari penulis.

Query parameters

category (optional)
Query Parameter — mengambil data sesuai kategori
searchString (optional)
Query Parameter — Mencari buku dengan mamasukan string nama bukunya atau nama penulis
skip (optional)
Query Parameter — number of records to skip for pagination format: int32
limit (optional)
Query Parameter — maximum number of records to return format: int32

Return type

BooksResponse

Example data

Content-Type: application/json
{
  "code" : 200,
  "data" : [ {
    "price" : 200000,
    "id" : 1,
    "title" : "Javascript 100jam",
    "category" : "Programming",
    "type" : "Free",
    "autor" : "Handoko"
  }, {
    "price" : 200000,
    "id" : 1,
    "title" : "Javascript 100jam",
    "category" : "Programming",
    "type" : "Free",
    "autor" : "Handoko"
  } ],
  "message" : "Success"
}

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

search results matching criteria or data book_types BooksResponse

respon ketika terjadi masalah di server
